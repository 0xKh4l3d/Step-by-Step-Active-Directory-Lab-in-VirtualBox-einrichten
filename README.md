# Dein-eigenes-AD-Lab-Von-der-Installation-zur-Benutzerverwaltung
Dieses Projekt richtet eine vollständige Microsoft-Domänenumgebung in Oracle VirtualBox ein: Windows Server 2022 als Domänencontroller (AD DS und DNS), RRAS/NAT für Internetzugang im Lab, DHCP für Clients sowie einen zusätzlichen Windows-11-Client zur praktischen Demonstration.
